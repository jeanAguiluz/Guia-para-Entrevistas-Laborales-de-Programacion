# 🎨 Desafíos de Frontend UI

Esta sección reúne ejercicios prácticos de interfaces para entrevistas técnicas de frontend. El objetivo es practicar maquetación, accesibilidad, estados de componentes y adaptación responsive.

---

## 📌 Qué vas a encontrar aquí

- Réplicas de pantallas comunes en procesos de selección.
- Componentes interactivos con foco en accesibilidad.
- Ejercicios de layout responsive y consistencia visual.
- Criterios de evaluación para revisar tu propia solución.

---

## 🛠️ Próximamente

1. Navbar responsive con menú móvil.
2. Modal accesible con manejo de foco.
3. Dashboard con cards, filtros y estados vacíos.
4. Formulario con validación y feedback visual.

---

## ✅ Cómo practicar

1. Replica la interfaz sin mirar una solución de referencia.
2. Prioriza HTML semántico y estilos mantenibles.
3. Verifica navegación por teclado y responsive design.
4. Explica tus decisiones como si estuvieras en una entrevista.
